<h2>training Project</h2>
<h3>web-customer-tracker</h3>
<br>

<i>Java 8, Maven, Hibernate, Spring MVC, Spring-AOP, Spring-Security</i><br><br>
1. Login user on the start page
2. Lists customers on the page.
<br>
<ul>
 <li>Role Employee - can see the list of customers and search for a customer</li>
 <li>Role Manager - can update customer</li>
 <li>Role Admin - can delete customers</li>
 </ul>
<br>
3. The list is sorted by last name.
<br>
4. The application allows you to add / edit / delete customers.
<br>
5. Search by first name or last name has been implemented. Also by name.
<br>
6. Add Logging with Spring-AOP
<br>
7. Register new user, as Employee
